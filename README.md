## About

VaRcade is a totally customizable open-source arcade room, designed to let you build **your own** arcade. Your favorite retro consoles, games and decorations, with realistic graphics and physics, all powered by RetroArch's emulation.

Everything you see is built with the exact same tools you get. A **room editor** to shape your space into your dream arcade, and a **console editor** to recreate any console you love, or invent ones that never existed. The consoles shipped with the game are made with the very editor you have in your hands.

Don't feel like building? Browse the **community section** in the spawn menu to find rooms, consoles, decorations and original creations made by other players.

VaRcade is in early development. Flat-screen Linux support comes first, with VR, Windows and standalone headsets to follow.

## Building from source

**Requirements**

- Unity 6.0 LTS (6000.0.75f1)
- RetroArch

```bash
git clone https://github.com/CyberRaybolt/VaRcade.git
```

Open the project folder in Unity Hub.
